---
title: "Ping Domain"
api: "GET https://xylex.dev/api/domains/ping"
---

<Card title="Endpoint Description" icon="info" horizontal>
  Executes a ping request to a specified domain to check its reachability. Requires a valid `domain` as a query parameter.
</Card>

<ParamField
  query="domain"
  type="String"
  required
  children="The domain name to ping (e.g., `app.xylex.ai`)."
>
  The domain name to ping.
</ParamField>
