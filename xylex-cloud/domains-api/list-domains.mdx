title: "List domains"
api: "GET https://xylex.dev/api/domains"
---

<Card title="Endpoint Description" icon="info" horizontal>
  Fetches a list of domains associated with the specified organization. Requires
  a valid `organization_id` as a query parameter.
</Card>

<ParamField
  query="organization_id"
  type="String"
  required
  children="The unique identifier of the organization whose domains are being queried."
>
  The organization ID to filter domains.
</ParamField>
