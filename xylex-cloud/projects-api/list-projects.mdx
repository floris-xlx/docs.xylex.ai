---
title: 'List Projects'
api: 'GET https://xylex.dev/api/projects'
---

<Card title="Endpoint Description" icon="info" horizontal>
Fetches all global projects for an organization, regardless of their status. This endpoint does not require any parameters and returns a list of projects with their respective statuses.
</Card>

### Possible Project Statuses
- **offline**: The project is currently not active.
- **errored**: The project encountered an error and is not functioning correctly.
- **stopped**: The project has been intentionally stopped.
- **online**: The project is active and running.
