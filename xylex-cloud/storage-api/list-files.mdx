---
title: "List Files"
api: "GET https://xylex.dev/api/files/list"
---

<Card title="Endpoint Description" icon="info" horizontal>
  Fetches a list of files associated with the specified organization and
  project. Requires valid `organization_id` and `project_id` as query
  parameters.
</Card>

<ParamField
  query="organization_id"
  type="String"
  required
  children="The organization ID to filter files by (e.g., `xlx-o-ea5a1270-7b03-11ef-b14f-325096b39f47`)."
>
  The organization ID to filter files.
</ParamField>

<ParamField
  query="project_id"
  type="String"
  required
  children="The project ID to filter files by (e.g., `1`)."
>
  The project ID to filter files.
</ParamField>
