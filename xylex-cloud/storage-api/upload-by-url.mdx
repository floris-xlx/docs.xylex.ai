---
title: "Upload File by URL"
api: "POST https://xylex.dev/api/files/upload/url"
---

<Card title="Endpoint Description" icon="info" horizontal>
  Uploads a file to the specified project and organization using a file URL. Requires valid `organization_id`, `project_id`, and `file_url` as query parameters.
</Card>

<ParamField
  query="organization_id"
  type="String"
  required
>
  The organization ID to associate the uploaded file with (e.g., `xlx-o-ea5a1270-7b03-11ef-b14f-325096b39f47`).
</ParamField>

<ParamField
  query="project_id"
  type="String"
  required
>
  The project ID to associate the uploaded file with (e.g., `1`).
</ParamField>

<ParamField
  query="file_url"
  type="String"
  required
>
  The URL of the file to upload (e.g., `https://xylex.com/gais.mp4`).
</ParamField>
