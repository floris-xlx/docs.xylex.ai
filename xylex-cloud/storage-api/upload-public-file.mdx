---
title: "Uploads a public file"
api: "POST https://xylex.cloud/api/files/upload-legacy"
---

<Card title="Endpoint Description" icon="info" horizontal>
  Uploads an unsigned public file
</Card>

<ParamField query="project_id" type="String" required>
  `project_id` that's targeted
</ParamField>
<ParamField query="organization_id" type="String" required>
  `organization_id` that's targeted
</ParamField>

<ParamField query="dir" type="String" required>
  relative `dir` that's targeted
</ParamField>

<ParamField query="return_url" type="Boolean" required>
  This will force return a url of the uploaded file
</ParamField>

Pass a file in a `formData` object

## Example response body

```json
{
  "uploadedFiles": [
    {
      "file_url": "https://ams3.digitaloceanspaces.com/xylex-cloud/system/integrations/revolut_personal.PNG"
    }
  ]
}
```
