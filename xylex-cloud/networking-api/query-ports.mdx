---
title: "Query Open Ports"
api: "GET https://xylex.dev/api/network/ports"
---

<Card title="Endpoint Description" icon="info" horizontal>
  Retrieves a list of open ports on the network. Supports filtering by protocol and port number using query parameters.
</Card>

<ParamField
  query="protocol"
  type="String"
  required={false}
  children="The protocol to filter ports by (e.g., `tcp`, `udp`)."
>
  The protocol to filter ports.
</ParamField>

<ParamField
  query="port"
  type="Number"
  required={false}
  children="The port number to filter by."
>
  The port number to filter ports.
</ParamField>
