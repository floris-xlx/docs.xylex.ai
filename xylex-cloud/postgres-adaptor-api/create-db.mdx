---
title: "Create Postgres DB and Setup PostgREST Wrapper"
api: "POST https://xylex.dev/api/db/postgresql/setup"
---

<Card title="Endpoint Description" icon="info" horizontal>
  Creates a new Postgres database and sets up a PostgREST wrapper for it. Requires a valid `database` name in the request body.
</Card>

<ParamField
  body="database"
  type="String"
  required
  children="The name of the database to create (e.g., `test1`)."
>
  The name of the database to create.
</ParamField>

<ResponseExample 
```json Response
{
    "success": false,
    "status": 422,
    "database_name": "test1",
    "error": "Database test1 already exists",
    "code": 422
}
```
</ResponseExample>

<ResponseExample 
```json Response
{
    "success": false,
    "status": 403,
    "database_name": "test2",
    "error": "Permission denied while creating the database.",
    "code": 403
}
```
</ResponseExample>
<ResponseExample 
```json Response
{
    "success": true,
    "status": 200,
    "message": "successfully created test2 database.",
    "database_name": "test1",
    "code": 200
}
```
</ResponseExample>
