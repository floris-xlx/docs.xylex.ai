### Yaml variables
```yaml
project:
  name: 'xylex.cloud'
  repository: 'https://github.com/floris-xlx/xylex.dev/settings'

server:
  port: 3034

auth:
  route_after_login: '/account'
````

<Card title="Awaiting deprecation" icon="exclamation" horizontal>
  Due to an automatic port negotiation controller this will become obsolete and
  be handled automatically
</Card>


<Card title=`server.port (number)`>
This defines what port the `next-server` runs which is then routed thru our `Proxy`   
</Card>
<Card title=`auth.route_after_login (string = '/account')`>
This defines where the user is sent after an authorizational state is gotten, defaults to `/account`
</Card>
