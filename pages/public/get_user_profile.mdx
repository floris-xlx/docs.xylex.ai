<Card title="Endpoint Description" icon="info" horizontal >Fetches a user`s profile by username, checks cache for existing data, and retrieves from the database if not cached. Returns user profile data with health status and response time. Caches the profile if fetched from the database.</Card>





<ParamField query="username" type="String" required  children="This parameter is used to specify the username of the user whose profile is being requested. It is required to uniquely identify the user in the system.">
  The username of the user to fetch the profile for.
</ParamField>
