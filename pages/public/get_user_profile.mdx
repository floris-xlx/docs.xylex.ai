<Card title="Endpoint Description" icon="info" horizontal >Fetches a user profile by username, first checking a cache, then querying a database if necessary, and returns the profile data and response time in milliseconds. If no profile is found, it returns an error message.</Card>





<ParamField query="username" type="String" required  children="The username parameter is used to identify and fetch the profile of a specific user. It is required for querying the user profile from the cache or database.">
  The username of the user.
</ParamField>
