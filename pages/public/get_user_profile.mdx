<Card title="Endpoint Description" icon="info" horizontal >Fetches a user`s profile by username, checks cache first, then queries the database if necessary. Returns profile data with a status and time taken for the operation. If not found, returns an error response.</Card>





<ParamField query="username" type="String" required  children="This parameter represents the username of the user whose profile information is being requested. It is used to fetch data from the cache or the database.">
  The username of the user whose profile is being fetched.
</ParamField>
