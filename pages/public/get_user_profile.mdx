<Card title="Endpoint Description" icon="info" horizontal >Fetches user profile by username, checks cache first, queries database if not cached, and stores result back in cache. Returns JSON response with status, username, profile data, and processing time.</Card>





<ParamField query="username" type="String" required  children="This parameter is used to identify and retrieve the specific user profile data associated with the provided username. It is essential for querying the user profile from both the cache and the database.">
  The username of the user to fetch the profile for.
</ParamField>
