title: 'User'

---



<Card title="Endpoint Description" icon="info" horizontal >Fetches a user profile using a username query, first checking a cache and then querying a database if necessary. Returns JSON with profile data, status, and response time. If no profile is found, it responds with an error message.</Card>





<ParamField query="username" type="String" required  children="This parameter represents the username of the user whose profile needs to be fetched. It is required for querying the user profile from the database or cache.">
  The username to fetch the user's profile.
</ParamField>
