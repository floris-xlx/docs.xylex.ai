<Card title="Endpoint Description" icon="info" horizontal >Fetches a user`s profile by username, checks cache first, and fetches from database if not cached. Returns JSON response with status, username, data, and time taken. Caches the profile if fetched from database.</Card>





<ParamField query="username" type="String" required  children="The username of the user whose profile is being requested. This is used to identify and retrieve the user's profile data from the cache or database.">
  The username to fetch the user profile for.
</ParamField>
