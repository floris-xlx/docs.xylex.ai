<Card title="Endpoint Description" icon="info" horizontal >Fetches a user profile by username, checking a cache first and then querying the database if necessary. Returns user data with status and time taken. Caches new profiles for future requests.</Card>





<ParamField query="username" type="String" required  children="The username parameter is used to identify the specific user whose profile is being fetched. It is a required parameter that must be provided in the query.">
  The username of the user.
</ParamField>
