<Card title="Endpoint Description" icon="info" horizontal >Fetches a user profile by username, checking cache first; if not found, retrieves from the database. Returns profile data with a status and time taken, caching the result for future requests.</Card>





<ParamField query="username" type="String" required  children="This parameter is used to identify the user whose profile information is requested from the database or cache. It is a mandatory parameter provided by the client in the request query.">
  The username of the user whose profile is being fetched.
</ParamField>
