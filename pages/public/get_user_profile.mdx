<Card title="Endpoint Description" icon="info" horizontal >Fetches a user`s profile by username, checks a cache first, and if not found, queries the database. Responds with user data, execution time, and status.</Card>





<ParamField query="username" type="String" required  children="The username parameter is a string that uniquely identifies the user in the system for whom the profile information needs to be retrieved. It is used to check the cache and if not found, query the database for the user's profile data.">
  The username of the user whose profile is to be fetched.
</ParamField>
