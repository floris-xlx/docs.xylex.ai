<Card title="Endpoint Description" icon="info" horizontal >Checks if a given strategy name is available. Logs the strategy name being checked. Returns a JSON response with the strategy name and its availability status.</Card>





<ParamField query="strategy_name" type="String" required  children="The strategy_name parameter is used to specify the name of the strategy that needs to be checked for its availability in the system. It should be a valid string representing the strategy.">
  Name of the strategy to check for availability.
</ParamField>
