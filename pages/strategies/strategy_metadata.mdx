<Card title="Endpoint Description" icon="info" horizontal >Fetches strategy metadata based on the provided organization ID. Logs the process and returns the metadata with a status indicating availability. Responds with a JSON object containing the organization, status, and data.</Card>





<ParamField query="strategy_id" type="String" required  children="This parameter is used to fetch metadata associated with a specific strategy identified by this unique identifier.">
  The unique identifier for a strategy.
</ParamField>
<ParamField query="organization" type="String" required  children="This parameter is used to fetch metadata associated with strategies belonging to this specific organization identified by this unique identifier.">
  The unique identifier for an organization.
</ParamField>
