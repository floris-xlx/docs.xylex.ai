<Card title="Endpoint Description" icon="info" horizontal >Fetches metadata for a strategy using a given strategy ID. Returns a JSON response with the status, strategy ID, and associated metadata if available. Logs the request for fetching strategy metadata by ID.</Card>





<ParamField query="strategy_id" type="String" required  children="This parameter represents the unique identifier for the strategy whose metadata is being fetched.">
  The ID of the strategy.
</ParamField>
<ParamField query="organization" type="String" required  children="This parameter represents the unique identifier or name of the organization whose strategy metadata is being fetched.">
  The name or ID of the organization.
</ParamField>
