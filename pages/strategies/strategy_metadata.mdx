---
title: 'Fetch Strategy Metadata By Strategy Id'
api: 'GET https://api.v2.xylex.ai/strategies/fetch_strategy_metadata_by_strategy_id'
---



<Card title="Endpoint Description" icon="info" horizontal >Fetches metadata for a specific strategy using a given strategy ID. Logs the retrieval process and checks if metadata is available, responding with its status and data.</Card>





<ParamField query="strategy_id" type="String" required  children="This parameter is a string that represents the unique identifier of the strategy for which metadata needs to be fetched. It is used to query the database and retrieve the relevant strategy metadata.">
  The ID of the strategy to fetch metadata for.
</ParamField>
<ParamField query="organization" type="String" required  children="This parameter is a string that represents the name or identifier of the organization for which strategy metadata needs to be fetched. It is used to query the database and retrieve the relevant metadata for all strategies associated with the specified organization.">
  The organization to fetch strategy metadata for.
</ParamField>
