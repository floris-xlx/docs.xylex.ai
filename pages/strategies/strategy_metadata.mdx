<Card title="Endpoint Description" icon="info" horizontal >Fetches strategy metadata based on the organization ID provided in the query. Logs the organization ID and checks the database for metadata. Responds with the metadata and a status indicating availability.</Card>





<ParamField query="strategy_id" type="String" required  children="This parameter is used to fetch metadata associated with a specific strategy by providing its unique identifier.">
  The unique identifier for a strategy.
</ParamField>
<ParamField query="organization" type="String" required  children="This parameter is used to fetch metadata associated with strategies that belong to a specific organization.">
  The organization identifier.
</ParamField>
