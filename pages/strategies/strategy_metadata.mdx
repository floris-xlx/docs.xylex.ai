---
title: 'Fetch Strategy Metadata By Strategy Id'
api: 'GET https://api.v2.xylex.ai/strategies/fetch_strategy_metadata_by_strategy_id'
---



<Card title="Endpoint Description" icon="info" horizontal >Fetches metadata for a strategy using a given strategy ID and returns a JSON response containing the status and the fetched data.</Card>





<ParamField query="strategy_id" type="String" required  children="This parameter represents the unique identifier of the strategy whose metadata is being fetched.">
  The ID of the strategy.
</ParamField>
<ParamField query="organization" type="String" required  children="This parameter is used to identify the organization for which the strategy metadata is being fetched.">
  The name of the organization.
</ParamField>
