<Card title="Endpoint Description" icon="info" horizontal >Fetches and computes statistics for a specific trading strategy using its ID. If successful, it returns the computed statistics; otherwise, it returns an error message.</Card>





<ParamField query="strategy_id" type="String" required  children="The unique identifier for the strategy for which statistics are being fetched or updated.">

</ParamField>
