<Card title="Endpoint Description" icon="info" horizontal >Fetches equity P&L data for a specified strategy ID over a given time period, defaulting to 12 months. Computes P&L panel data and returns it along with the strategy`s health status.</Card>





<ParamField query="strategy_id" type="String" required  children="The `strategy_id` parameter is a unique string that identifies the specific strategy for which the equity PnL (Profit and Loss) data is being fetched. It is mandatory and used to query the relevant data.">
  Unique identifier for the strategy.
</ParamField>
<ParamField query="data_length" type="Option<String>"  children="The `data_length` parameter specifies the optional time period for which the equity PnL data should be computed. It can take values such as '12_months', '30_days', '7_days', or '24_hours'. If not provided, it defaults to '12_months'.">
  Optional time period for data retrieval.
</ParamField>
