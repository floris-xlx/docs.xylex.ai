<Card title="Endpoint Description" icon="info" horizontal >Fetches equity PnL data for a specified strategy using its ID and optional time range. Returns JSON with status, strategy ID, and data. Defaults to 12-month data if no range is specified.</Card>





<ParamField query="strategy_id" type="String" required  children="A string that uniquely identifies the strategy whose equity PNL data is being fetched.">
  The unique identifier for a strategy
</ParamField>
<ParamField query="data_length" type="Option<String>"  children="An optional string that specifies the length of time for which the equity PNL data is to be fetched. Defaults to '12_months' if not provided.">
  The time period for data retrieval
</ParamField>
