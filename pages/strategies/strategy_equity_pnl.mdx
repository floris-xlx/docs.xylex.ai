<Card title="Endpoint Description" icon="info" horizontal >Fetches the equity profit and loss data for a given strategy ID over a specified time period, defaulting to 12 months. Returns the data along with a health status (`Healthy` or `Unhealthy`).</Card>





<ParamField query="strategy_id" type="String" required  children="A unique identifier for the strategy whose equity PnL data is being fetched.">
  The ID of the strategy.
</ParamField>
<ParamField query="data_length" type="Option<String>"  children="An optional string representing the desired time period for which the equity PnL data should be fetched. Defaults to '12_months' if not provided.">
  The length of the data period.
</ParamField>
