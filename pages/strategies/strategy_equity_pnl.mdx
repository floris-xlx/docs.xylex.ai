<Card title="Endpoint Description" icon="info" horizontal >Fetches equity PnL data for a given strategy ID over a specified time period, defaulting to 12 months if not provided, and returns the data along with a health status.</Card>





<ParamField query="strategy_id" type="String" required  children="The unique identifier for the strategy whose equity PnL data is to be fetched.">
  Strategy ID
</ParamField>
<ParamField query="data_length" type="Option<String>"  children="The optional parameter that specifies the length of data to fetch, such as '12_months', '30_days', etc.">
  Data Length
</ParamField>
