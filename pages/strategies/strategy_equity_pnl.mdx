---
title: 'Equity Pnl'
api: 'GET https://api.v2.xylex.ai/strategies/equity_pnl'
---



<Card title="Endpoint Description" icon="info" horizontal >This endpoint retrieves the equity PnL data for a given strategy by its ID, over a specified time period. It defaults to 12 months if no duration is provided and returns a JSON response with status and trade data.</Card>





<ParamField query="strategy_id" type="String" required  children="A string representing the unique identifier for the strategy, which is used to fetch the equity PnL data associated with this specific strategy.">
  The unique identifier for the strategy.
</ParamField>
<ParamField query="data_length" type="Option<String>"  children="An optional string that specifies the length of historical data to fetch for the strategy. If not provided, defaults to '12_months'.">
  The length of data to fetch.
</ParamField>
