<Card title="Endpoint Description" icon="info" horizontal >Creates a strategy name unique to an organization. Accepts `strategy_name`, `strategy_id`, and `organization_id` as parameters. Responds with the strategy name and a creation status.</Card>





<ParamField query="strategy_name" type="String" required  children="The strategy_name parameter represents the unique name of the strategy that is being created within an organization.">
  Name of the strategy.
</ParamField>
<ParamField query="strategy_id" type="String" required  children="The strategy_id parameter represents a unique identifier for the strategy being created. This ID helps in distinguishing different strategies within the system.">
  Identifier for the strategy.
</ParamField>
<ParamField query="organization_id" type="String" required  children="The organization_id parameter represents the unique identifier for the organization within which the strategy is being created. This ensures that strategies are unique to each organization.">
  Identifier for the organization.
</ParamField>
