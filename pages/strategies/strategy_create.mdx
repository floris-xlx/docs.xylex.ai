---
title: 'Name Create'
api: 'POST https://api.v2.xylex.ai/strategy/name_create'
---



<Card title="Endpoint Description" icon="info" horizontal >Creates a unique strategy name for an organization using provided strategy name, strategy ID, and organization ID. Checks uniqueness before adding to the database. Returns the strategy name and a creation status.</Card>





<ParamField query="strategy_name" type="String" required  children="This parameter represents the name of the strategy that is to be created. It must be unique within the organization.">
  The name of the strategy.
</ParamField>
<ParamField query="strategy_id" type="String" required  children="This parameter represents the unique identifier for the strategy. It is used within the system to distinguish between different strategies.">
  The ID of the strategy.
</ParamField>
<ParamField query="organization_id" type="String" required  children="This parameter represents the unique identifier for the organization that owns the strategy. It ensures that strategies are unique within the given organization.">
  The ID of the organization.
</ParamField>
