---
title: 'Name Create'
api: 'POST https://api.v2.xylex.ai/strategy/name_create'
---



<Card title="Endpoint Description" icon="info" horizontal >Creates a strategy name unique to an organization using provided strategy name, strategy ID, and organization ID. Logs the process and checks database uniqueness before creation. Returns JSON with the strategy name and creation status.</Card>





<ParamField query="strategy_name" type="String" required  children="This parameter represents the name of the strategy and must be unique within the organization. It is required to create a new strategy entry.">
  The name of the strategy.
</ParamField>
<ParamField query="strategy_id" type="String" required  children="This parameter represents the unique identifier for the strategy and is required for creating a new strategy entry.">
  The identifier of the strategy.
</ParamField>
<ParamField query="organization_id" type="String" required  children="This parameter represents the unique identifier for the organization and is required to ensure the strategy name is unique within the organization.">
  The identifier of the organization.
</ParamField>
