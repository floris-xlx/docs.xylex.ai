title: 'Name Create'
api: 'POST https://api.v2.xylex.ai/strategy/name_create'
---



<Card title="Endpoint Description" icon="info" horizontal >Creates a unique strategy name for an organization using the provided strategy name, strategy ID, and organization ID. Returns a JSON response indicating if the strategy name was successfully created.</Card>





<ParamField query="strategy_name" type="String" required  children="The name of the strategy to be created, which must be unique to the organization.">
  Name of the strategy
</ParamField>
<ParamField query="strategy_id" type="String" required  children="A unique identifier for the strategy being created.">
  ID of the strategy
</ParamField>
<ParamField query="organization_id" type="String" required  children="A unique identifier for the organization, ensuring that the strategy name is unique within this context.">
  ID of the organization
</ParamField>
