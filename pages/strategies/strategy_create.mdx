title: 'Name Create'
api: 'POST https://api.v2.xylex.ai/strategy/name_create'
---



<Card title="Endpoint Description" icon="info" horizontal >Creates a unique strategy name for an organization, verifying its uniqueness before adding it to the database. Accepts `strategy_name`, `strategy_id`, and `organization_id` as query parameters. Returns the strategy name and its creation status in JSON format.</Card>





<ParamField query="strategy_name" type="String" required  children="The name of the strategy, which must be unique to the organization.">
  The name of the strategy.
</ParamField>
<ParamField query="strategy_id" type="String" required  children="The unique identifier for the strategy.">
  The ID of the strategy.
</ParamField>
<ParamField query="organization_id" type="String" required  children="The unique identifier for the organization.">
  The ID of the organization.
</ParamField>
