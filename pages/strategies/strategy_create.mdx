<Card title="Endpoint Description" icon="info" horizontal >Creates a strategy name unique to the organization using provided parameters: strategy_name, strategy_id, and organization_id. Checks uniqueness and adds it to the database. Returns JSON with the strategy name and creation status.</Card>





<ParamField query="strategy_name" type="String" required  children="This parameter represents the name of the strategy. It must be unique to the organization.">
  The name of the strategy.
</ParamField>
<ParamField query="strategy_id" type="String" required  children="This parameter serves as the unique identifier for the strategy within the system.">
  The identifier for the strategy.
</ParamField>
<ParamField query="organization_id" type="String" required  children="This parameter is the unique identifier for the organization to which the strategy belongs.">
  The organization identifier.
</ParamField>
