<Card title="Endpoint Description" icon="info" horizontal >Creates a unique strategy name for an organization by accepting strategy name, strategy ID, and organization ID as parameters. Validates uniqueness and records in the database. Returns the created strategy name and a success status.</Card>





<ParamField query="strategy_name" type="String" required  children="This parameter represents the unique name of the strategy within the organization that is to be created.">
  Name of the strategy to be created.
</ParamField>
<ParamField query="strategy_id" type="String" required  children="This parameter is a unique identifier for the strategy being created and is required for tracking and managing the strategy.">
  Identifier for the strategy.
</ParamField>
<ParamField query="organization_id" type="String" required  children="This parameter identifies the organization to which the strategy belongs, ensuring the strategy name is unique within this organization.">
  Identifier for the organization.
</ParamField>
