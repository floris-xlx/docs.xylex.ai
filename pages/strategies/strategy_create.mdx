title: 'Name Create'
api: 'POST https://api.v2.xylex.ai/strategy/name_create'
---



<Card title="Endpoint Description" icon="info" horizontal >Creates a strategy name unique to the organization. Accepts `strategy_name`, `strategy_id`, and `organization_id` as query parameters. Returns a JSON response indicating whether the strategy name was successfully created.</Card>





<ParamField query="strategy_name" type="String" required  children="The strategy_name parameter specifies the name of the strategy to be created. It is required and must be unique to the organization.">
  The name of the strategy.
</ParamField>
<ParamField query="strategy_id" type="String" required  children="The strategy_id parameter identifies the strategy to be created. It is required and must be unique within the organization.">
  The ID of the strategy.
</ParamField>
<ParamField query="organization_id" type="String" required  children="The organization_id parameter specifies the organization to which the strategy belongs. It is required and ensures that the strategy name is unique within this organization.">
  The ID of the organization.
</ParamField>
