---
title: 'Name Create'
api: 'POST https://api.v2.xylex.ai/strategy/name_create'
---



<Card title="Endpoint Description" icon="info" horizontal >Creates a unique strategy name for an organization by checking its uniqueness and adding it to the database. Returns a JSON response indicating the strategy name and whether it was successfully created.</Card>





<ParamField query="strategy_name" type="String" required  children="The strategy name is a required parameter that is unique to the organization and will be used to create a strategy name in the database.">
  The name of the strategy to be created.
</ParamField>
<ParamField query="strategy_id" type="String" required  children="The strategy ID is a required parameter that serves as a unique identifier for the strategy being created within the organization.">
  The identifier for the strategy.
</ParamField>
<ParamField query="organization_id" type="String" required  children="The organization ID is a required parameter that identifies the organization to which the strategy belongs.">
  The identifier for the organization.
</ParamField>
