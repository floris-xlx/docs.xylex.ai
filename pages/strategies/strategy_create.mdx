<Card title="Endpoint Description" icon="info" horizontal >Creates a strategy name for an organization, ensuring uniqueness. Takes `strategy_name`, `strategy_id`, and `organization_id` as parameters. Responds with success status and the strategy name.</Card>





<ParamField query="strategy_name" type="String" required  children="This parameter represents the name of the strategy, which must be unique within the organization.">
  The name of the strategy.
</ParamField>
<ParamField query="strategy_id" type="String" required  children="This parameter represents the unique identifier of the strategy.">
  The ID of the strategy.
</ParamField>
<ParamField query="organization_id" type="String" required  children="This parameter represents the unique identifier of the organization to which the strategy belongs.">
  The ID of the organization.
</ParamField>
