<Card title="Endpoint Description" icon="info" horizontal >Creates a unique strategy name for an organization using provided `strategy_name`, `strategy_id`, and `organization_id`. Checks database for name uniqueness and responds with creation status.</Card>





<ParamField query="strategy_name" type="String" required  children="This parameter represents the name of the strategy, which must be unique within the organization.">
  The name of the strategy.
</ParamField>
<ParamField query="strategy_id" type="String" required  children="This parameter is used to uniquely identify the strategy within the system.">
  The identifier for the strategy.
</ParamField>
<ParamField query="organization_id" type="String" required  children="This parameter is used to specify the organization to which the strategy belongs, ensuring the strategy name's uniqueness within this context.">
  The identifier for the organization.
</ParamField>
