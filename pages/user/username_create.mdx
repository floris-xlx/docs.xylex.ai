---
title: 'Name Create'
api: 'POST https://api.v2.xylex.ai/user/name_create'
---



<Card title="Endpoint Description" icon="info" horizontal >Checks if a username is unique, attempts to create it in the database, and returns a JSON response indicating the outcome.</Card>





<ParamField query="username" type="String" required  children="This parameter represents the username that the user wishes to create. It must be unique and will be checked against the database to ensure it is not already in use.">
  The desired username to create.
</ParamField>
<ParamField query="user_id" type="String" required  children="This parameter represents the unique identifier of the user in the system who is attempting to create a new username. It is used to associate the username with the correct user.">
  The ID of the user attempting to create the username.
</ParamField>
