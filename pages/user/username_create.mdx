---
title: 'Name Create'
api: 'POST https://api.v2.xylex.ai/user/name_create'
---



<Card title="Endpoint Description" icon="info" horizontal >Creates a unique username for a user, requiring `username` and `user_id` as query parameters, and returns a JSON response indicating if the username was successfully created.</Card>





<ParamField query="username" type="String" required  children="The desired username that the user wants to create, which must be unique.">
  Username to be created.
</ParamField>
<ParamField query="user_id" type="String" required  children="The unique identifier of the user for whom the username is being created.">
  ID of the user.
</ParamField>
