---
title: 'Name Create'
api: 'POST https://api.v2.xylex.ai/user/name_create'
---



<Card title="Endpoint Description" icon="info" horizontal >Creates a unique username for a user by checking its availability and adding it to the database, returning a JSON response with the username and a creation status.</Card>





<ParamField query="username" type="String" required  children="The 'username' parameter represents the desired name that the user wants to create. It must be unique and will be checked against existing records before creation.">
  The desired username to be created.
</ParamField>
<ParamField query="user_id" type="String" required  children="The 'user_id' parameter is a unique identifier for the user who is attempting to create a new username. It is used to associate the username with the correct user account.">
  The identifier of the user requesting the username.
</ParamField>
