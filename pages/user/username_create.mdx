---
title: 'Name Create'
api: 'POST https://api.v2.xylex.ai/user/name_create'
---



<Card title="Endpoint Description" icon="info" horizontal >Creates a username for a user, ensuring its uniqueness, using the provided `username` and `user_id` parameters. Returns a JSON response indicating the username and whether it was successfully created.</Card>





<ParamField query="username" type="String" required  children="A unique string representing the username that is intended to be created for the user.">
  The username to be created.
</ParamField>
<ParamField query="user_id" type="String" required  children="A string representing the unique identifier of the user for whom the username is being created.">
  The ID of the user.
</ParamField>
