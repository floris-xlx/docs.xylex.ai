---
title: 'Name Create'
api: 'POST https://api.v2.xylex.ai/user/name_create'
---



<Card title="Endpoint Description" icon="info" horizontal >Creates a username for a user by checking its uniqueness and adding it to the database. Accepts `username` and `user_id` as required query parameters. Returns a JSON response indicating the username and whether it was successfully created.</Card>





<ParamField query="username" type="String" required  children="A string representing the new username that the user wishes to create. It must be unique.">
  The desired username for the user.
</ParamField>
<ParamField query="user_id" type="String" required  children="A string representing the unique ID of the user requesting to create the username.">
  The unique identifier for the user.
</ParamField>
