<Card title="Endpoint Description" icon="info" horizontal >Checks if a username is unique and adds it to the database, returning a JSON response indicating the status.</Card>





<ParamField query="username" type="String" required  children="This parameter represents the username that the user wants to create. It needs to be unique and follows any naming conventions or restrictions set by the application.">
  The desired username for creation.
</ParamField>
<ParamField query="user_id" type="String" required  children="This parameter is a unique identifier for the user who is attempting to create a new username. It is used to associate the new username with the correct user in the system.">
  The user's unique identifier.
</ParamField>
