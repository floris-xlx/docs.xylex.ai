---



<Card title="Endpoint Description" icon="info" horizontal >Creates a username for a user, requiring a `username` and `user_id`. Validates uniqueness and attempts to add it to the database. Responds with the username and a creation success status.</Card>





<ParamField query="username" type="String" required  children="The username parameter is a required string that represents the desired username that a user wants to create in the system. It must be unique and will be checked against existing usernames to ensure no duplicates.">
  The desired username for the user.
</ParamField>
<ParamField query="user_id" type="String" required  children="The user_id parameter is a required string that serves as a unique identifier for the user who is attempting to create a new username. This ID is used to associate the username with the correct user account in the system.">
  The unique identifier for the user.
</ParamField>
