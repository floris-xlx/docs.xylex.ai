---
title: 'Name Create'
api: 'POST https://api.v2.xylex.ai/user/name_create'
---



<Card title="Endpoint Description" icon="info" horizontal >Creates a username for a user, checking its uniqueness before adding it to the database, and returns whether the username was successfully created.</Card>





<ParamField query="username" type="String" required  children="This parameter is the new username that the user wishes to create. It should be unique and will be checked against the database for availability.">
  The desired username to be created.
</ParamField>
<ParamField query="user_id" type="String" required  children="This parameter is the unique identifier of the user who is attempting to create a new username. It is used to associate the username with the correct user account in the database.">
  The ID of the user making the request.
</ParamField>
