---
title: 'Name Create'
api: 'POST https://api.v2.xylex.ai/user/name_create'
---



<Card title="Endpoint Description" icon="info" horizontal >Creates a username associated with a user ID after checking its uniqueness in the database. Returns a JSON response indicating the username and whether creation was successful.</Card>





<ParamField query="username" type="String" required  children="Specifies the desired username that the user wants to create and add to the database. This parameter is checked for uniqueness before creation.">
  The desired username to be created.
</ParamField>
<ParamField query="user_id" type="String" required  children="Provides the unique identifier of the user attempting to create a new username, which is used to associate the username with the correct user in the database.">
  The ID of the user creating the username.
</ParamField>
