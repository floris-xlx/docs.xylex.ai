<Card title="Endpoint Description" icon="info" horizontal >Fetches trade metadata for a specific user based on the provided user ID. Logs the user ID and retrieves trade metadata from the database. Returns a JSON response with the status, user ID, and trade metadata.</Card>





<ParamField query="user_id" type="String" required  children="The user_id is a string parameter that uniquely identifies a user for fetching their trade metadata.">
  The user's unique identifier.
</ParamField>
