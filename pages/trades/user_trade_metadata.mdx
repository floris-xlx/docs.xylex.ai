<Card title="Endpoint Description" icon="info" horizontal >Fetches trade metadata for a user based on their user_id. Logs the user_id being fetched. Returns a JSON response with the status, user_id, and trade metadata.</Card>





<ParamField query="user_id" type="String" required  children="This parameter is used to identify and fetch trade metadata associated with a specific user. It is required for querying the user's trade data.">
  The unique identifier for a user.
</ParamField>
