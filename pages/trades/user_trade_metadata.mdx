<Card title="Endpoint Description" icon="info" horizontal >Fetches trade metadata for a specified user by user_id, returning a JSON response. Includes a status indicating if the metadata is available. Logs the user_id being queried.</Card>





<ParamField query="user_id" type="String" required  children="A unique identifier representing the user whose trade metadata is to be fetched.">
  The ID of the user.
</ParamField>
