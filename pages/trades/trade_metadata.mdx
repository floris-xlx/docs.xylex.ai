<Card title="Endpoint Description" icon="info" horizontal >Fetches trade metadata using a specified trade hash and returns it with a status indicating health.</Card>





<ParamField query="trade_hash" type="String" required  children="A unique identifier for the trade that is used to fetch the associated metadata from the database. It is required to identify which trade's metadata needs to be retrieved.">
  The hash of the trade to fetch metadata for.
</ParamField>
