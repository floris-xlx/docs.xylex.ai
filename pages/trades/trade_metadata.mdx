<Card title="Endpoint Description" icon="info" horizontal >Fetches trade metadata using a provided trade hash, returning the data and a status indicating `Healthy` if metadata is found, otherwise `Unhealthy`.</Card>





<ParamField query="trade_hash" type="String" required  children="The 'trade_hash' is a unique identifier used to fetch specific metadata associated with a particular trade. It is required for querying trade metadata from the data source.">
  The hash of the trade to fetch metadata for.
</ParamField>
