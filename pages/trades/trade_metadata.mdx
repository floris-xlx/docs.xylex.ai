<Card title="Endpoint Description" icon="info" horizontal >Fetches trade metadata using a provided trade hash, checks if the data exists, and returns a JSON response with the status (`Healthy` or `Unhealthy`), the trade hash, and the metadata.</Card>





<ParamField query="trade_hash" type="String" required  children="A unique hash string that identifies the trade for which metadata is being fetched.">
  Hash of the trade
</ParamField>
