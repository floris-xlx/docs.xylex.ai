<Card title="Endpoint Description" icon="info" horizontal >Fetches trade metadata based on a provided `trade_hash`. Logs the retrieval process and checks the database for the metadata. Returns a JSON response indicating the health status and the trade metadata.</Card>





<ParamField query="trade_hash" type="String" required  children="The trade_hash parameter is a unique identifier in the form of a string used to fetch specific trade metadata from the database.">
  The hash identifier for the trade.
</ParamField>
