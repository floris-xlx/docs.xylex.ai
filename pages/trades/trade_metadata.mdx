<Card title="Endpoint Description" icon="info" horizontal >Fetches trade metadata using a specified trade hash, logs the process, and returns the trade data with a health status.</Card>





<ParamField query="trade_hash" type="String" required  children="This parameter is used to identify the specific trade for which metadata is being fetched. It should be a valid string representing the trade's unique hash.">
  The hash of the trade to fetch metadata for.
</ParamField>
