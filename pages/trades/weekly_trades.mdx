<Card title="Endpoint Description" icon="info" horizontal >Fetches weekly trade data for a specific user and organization, verifying the user`s organization. Returns trades for the requested week, filtering out unapproved or invalid trades. Provides error responses for mismatches, missing data, or internal issues.</Card>





<ParamField query="user_id" type="String" required  children="A unique identifier for the user making the request.">
  User identifier
</ParamField>
<ParamField query="organization" type="String" required  children="The organization to which the user belongs and for which the trades are being requested.">
  Organization identifier
</ParamField>
<ParamField query="week_nr" type="Option<i64>"  children="The specific week number for which trades are being requested. If not provided, defaults to the current week.">
  Week number
</ParamField>
<ParamField query="algorithm_id" type="String" required  children="A unique identifier for the algorithm associated with the trades being requested.">
  Algorithm identifier
</ParamField>
