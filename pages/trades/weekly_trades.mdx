<Card title="Endpoint Description" icon="info" horizontal >Fetches trades for a specified organization and week. Requires `user_id` and `organization`, with an optional `week_nr`. Returns trades or error messages based on user verification and data retrieval.</Card>





<ParamField query="user_id" type="String" required  children="A unique identifier for the user making the request, required for trade retrieval and user verification.">
  The ID of the user.
</ParamField>
<ParamField query="organization" type="String" required  children="The name of the organization associated with the user, required for trade retrieval and verification.">
  The organization name.
</ParamField>
<ParamField query="week_nr" type="Option<i64>"  children="The specific week number for which trades are to be retrieved, optional and defaults to the current week if not provided.">
  The week number.
</ParamField>
<ParamField query="algorithm_id" type="String" required  children="A unique identifier for the algorithm, required for retrieving trades specific to that algorithm.">
  The ID of the algorithm.
</ParamField>
