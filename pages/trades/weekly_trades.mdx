title: 'Get Trades Weekly Org'
api: 'GET https://api.v2.xylex.ai/get_trades_weekly_org'
---



<Card title="Endpoint Description" icon="info" horizontal >Fetches weekly trades for a specified user, organization, and algorithm. Validates user organization and retrieves trades excluding `unapproved` or `invalid` statuses. Returns trades data or error details as a JSON response.</Card>





<ParamField query="user_id" type="String" required  children="The unique identifier for the user requesting the trades.">
  User identifier
</ParamField>
<ParamField query="organization" type="String" required  children="The name of the organization to which the user belongs or is querying trades for.">
  Organization name
</ParamField>
<ParamField query="week_nr" type="Option<i64>"  children="The specific week number for which the trades are requested; if not provided, defaults to the current week number.">
  Week number
</ParamField>
<ParamField query="algorithm_id" type="String" required  children="The unique identifier for the algorithm associated with the requested trades.">
  Algorithm identifier
</ParamField>
