<Card title="Endpoint Description" icon="info" horizontal >Fetches weekly trades for a specific algorithm, verifying user organization. Returns trades matching the given week number or current week, excluding unapproved or invalid trades. Validates user data and handles errors with appropriate HTTP responses.</Card>

</ParamField>

  Organization name
</ParamField>

<ParamField query="algorithm_id" type="String" required  children="A unique identifier for the algorithm used in the trade.">
