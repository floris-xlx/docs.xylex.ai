<Card title="Endpoint Description" icon="info" horizontal >Retrieves the status of a trade using a provided trade hash. Logs the trade hash and queries the database to get the trade status. Returns the trade hash and status in JSON format.</Card>





<ParamField query="trade_hash" type="String" required  children="This is the unique identifier for a trade, used to query and fetch the current status of that trade from the database.">
  Hash of the trade to fetch status for.
</ParamField>
