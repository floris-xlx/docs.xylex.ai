<Card title="Endpoint Description" icon="info" horizontal >Fetches the status of a trade using a provided trade hash. Logs the request and retrieves status from the database. Returns a JSON response with the trade hash and its status.</Card>





<ParamField query="trade_hash" type="String" required  children="A unique string representing the hash identifier of the trade, used to fetch the trade status from the database.">
  The hash identifier for the trade.
</ParamField>
