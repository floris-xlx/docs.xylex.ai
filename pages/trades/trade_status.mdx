<Card title="Endpoint Description" icon="info" horizontal >Fetches the status of a trade using a provided trade hash, logs the query, and returns the trade status in a JSON response.</Card>





<ParamField query="trade_hash" type="String" required  children="A unique identifier represented as a string that specifies the trade whose status is to be fetched.">
  The hash of the trade.
</ParamField>
