<Card title="Endpoint Description" icon="info" horizontal >Fetches and returns the status of a trade based on a provided trade hash. Logs the trade hash being checked. Responds with a JSON object containing the trade hash and its status.</Card>





<ParamField query="trade_hash" type="String" required  children="A unique string that serves as an identifier for the trade, which is used to fetch the status of the trade from the database.">
  The hash identifier for the trade.
</ParamField>
