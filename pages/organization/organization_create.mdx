<Card title="Endpoint Description" icon="info" horizontal >This endpoint checks if an organization name is unique and attempts to add it to the database. It accepts a query with `organization_name` and `organization_id`, returning a JSON response indicating the name and whether it was successfully created.</Card>





<ParamField query="organization_name" type="String" required  children="This parameter represents the name of the organization that is to be created or checked for uniqueness in the database.">
  Name of the organization.
</ParamField>
<ParamField query="organization_id" type="String" required  children="This parameter is the unique identifier for the organization, used to verify and manage the organization within the system.">
  ID of the organization.
</ParamField>
