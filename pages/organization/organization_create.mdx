<Card title="Endpoint Description" icon="info" horizontal >Checks if an organization name is unique and adds it to the database. Returns a JSON response indicating the organization name and whether it was successfully created.</Card>





<ParamField query="organization_name" type="String" required  children="This parameter specifies the desired name for the organization being created. It must be unique and will be checked against existing entries to ensure no duplicates.">
  The name of the organization to be created.
</ParamField>
<ParamField query="organization_id" type="String" required  children="This parameter represents a unique identifier assigned to the organization. It is used to differentiate between organizations, especially when names might be similar or identical.">
  The unique identifier for the organization.
</ParamField>
