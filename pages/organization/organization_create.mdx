title: 'Name Create'
api: 'POST https://api.v2.xylex.ai/organization/name_create'
---



<Card title="Endpoint Description" icon="info" horizontal >This endpoint checks if the provided organization name is unique and attempts to add it to the database. It requires the organization name and ID as query parameters. It responds with the organization name and a boolean indicating if the creation was successful.</Card>





<ParamField query="organization_name" type="String" required  children="The name of the organization that is to be created or checked for uniqueness.">
  Name of the organization.
</ParamField>
<ParamField query="organization_id" type="String" required  children="A unique identifier associated with the organization that is to be created or checked.">
  Identifier for the organization.
</ParamField>
