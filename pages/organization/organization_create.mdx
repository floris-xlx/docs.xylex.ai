---
title: 'Name Create'
api: 'POST https://api.v2.xylex.ai/organization/name_create'
---



<Card title="Endpoint Description" icon="info" horizontal >This endpoint checks if an organization name is unique, and if so, adds it to the database. It takes `organization_name` and `organization_id` as query parameters, and returns a JSON response indicating the name and whether it was successfully created.</Card>





<ParamField query="organization_name" type="String" required  children="This parameter represents the name of the organization that is being checked for uniqueness and potentially added to the database.">
  The name of the organization.
</ParamField>
<ParamField query="organization_id" type="String" required  children="This parameter is the unique identifier for the organization associated with the name being created or validated.">
  The ID of the organization.
</ParamField>
