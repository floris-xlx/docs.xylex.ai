<Card title="Endpoint Description" icon="info" horizontal >This endpoint checks if an organization name is unique and adds it to the database. It takes an organization name and ID as query parameters. It returns a JSON response indicating the organization name and whether it was successfully created.</Card>





<ParamField query="organization_name" type="String" required  children="This parameter represents the name of the organization that is being created. It must be unique and is used to identify the organization within the system.">
  The name of the organization.
</ParamField>
<ParamField query="organization_id" type="String" required  children="This parameter is used as a unique identifier for the organization. It is required to distinguish between different organizations within the database.">
  The unique identifier for the organization.
</ParamField>
