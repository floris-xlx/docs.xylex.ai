---
title: 'Name Create'
api: 'POST https://api.v2.xylex.ai/organization/name_create'
---



<Card title="Endpoint Description" icon="info" horizontal >Creates a new organization name if it is unique, associating it with a given organization ID, and returns a JSON response indicating the organization name and creation status.</Card>





<ParamField query="organization_name" type="String" required  children="The name of the organization that is intended to be created.">
  Name of the organization.
</ParamField>
<ParamField query="organization_id" type="String" required  children="A unique identifier for the organization that is intended to be created.">
  ID of the organization.
</ParamField>
