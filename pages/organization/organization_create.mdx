<Card title="Endpoint Description" icon="info" horizontal >Creates a new organization name if it`s unique, using provided name and ID, and returns a JSON response indicating success.</Card>





<ParamField query="organization_name" type="String" required  children="This parameter represents the name of the organization that is intended to be created. It must be provided in the request query and ensures that the organization name is recorded and checked for uniqueness before being added to the database.">
  The name of the organization to be created.
</ParamField>
<ParamField query="organization_id" type="String" required  children="This parameter is a unique identifier for the organization that is being created. It accompanies the organization name in the query to help verify and manage the organization's data within the system.">
  The unique identifier of the organization.
</ParamField>
