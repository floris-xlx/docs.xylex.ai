---
title: 'Time Series'
api: 'GET https://api.v2.xylex.ai/data/time_series'
---



<Card title="Endpoint Description" icon="info" horizontal >Fetches time series data for a specified symbol and interval, checking cache first, then an external API if needed. Requires a valid API key and supports optional start and end dates. Responds with data from cache or API, indicating the source.</Card>





<ParamField query="symbol" type="String" required  children="Represents the ticker symbol of the financial instrument for which the time series data is requested.">
  Ticker symbol for the financial instrument.
</ParamField>
<ParamField query="start_date" type="Option<String>"  children="Optional parameter specifying the start date from which the time series data should be retrieved. The format is expected to be in YYYY-MM-DD.">
  Start date for the time series data.
</ParamField>
<ParamField query="end_date" type="Option<String>"  children="Optional parameter specifying the end date until which the time series data should be retrieved. The format is expected to be in YYYY-MM-DD.">
  End date for the time series data.
</ParamField>
<ParamField query="interval" type="String" required  children="Represents the interval for the time series data such as daily, weekly, etc.">
  Interval for the time series data.
</ParamField>
<ParamField query="outputsize" type="u32" required  children="Specifies the number of data points to be retrieved in the time series data.">
  Size of the output data.
</ParamField>
<ParamField query="api_key" type="String" required  children="A key used to authenticate the request, ensuring that the client has the necessary permissions to access the API.">
  API key for authentication.
</ParamField>
