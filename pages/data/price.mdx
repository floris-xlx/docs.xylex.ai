<Card title="Endpoint Description" icon="info" horizontal >Fetches real-time price data for a given financial symbol. Requires a valid API key for access. Utilizes caching to improve response times for repeat requests.</Card>

</ParamField>
<ParamField query="api_key" type="String" required  children="The API key is a unique identifier used to authenticate the user requesting the real-time price data. It ensures that the request is authorized and prevents unauthorized access to the service.">

</ParamField>
