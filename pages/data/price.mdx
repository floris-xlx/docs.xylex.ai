<Card title="Endpoint Description" icon="info" horizontal >Fetches real-time price data for a specified symbol, requiring an API key for access. Checks for cached data and returns it if available; otherwise, retrieves data from an external API. Responds with JSON indicating success and the source of data.</Card>





<ParamField query="symbol" type="String" required  children="The stock symbol is a unique series of letters assigned to a security for trading purposes. This parameter is crucial for identifying the specific stock whose real-time price is to be retrieved.">
  The stock symbol for which the real-time price is requested.
</ParamField>
<ParamField query="api_key" type="String" required  children="The API key is a code passed in by the user to authenticate an application or user with an external system or API. This parameter is required to ensure that the requestor has permission to access the real-time pricing data.">
  The API key for authentication.
</ParamField>
